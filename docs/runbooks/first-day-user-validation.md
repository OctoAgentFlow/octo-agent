# First-Day User Validation Checklist

Status: Active.

Use this checklist when testing the first useful OctoAgentFlow session with a
new operator account. The goal is not to prove growth claims; it is to confirm
that a user can understand and complete the manual Daily Growth Desk loop.

## Test User Setup

- One X account connected or ready to connect.
- One OAF Bot created or generated from Account Intelligence.
- Optional: pasted website/product context or Creator Studio observations.
- No expectation of automatic commenting or automatic outreach.

## Success Path

1. Open `/start-today`.
2. Confirm the path points the user into Daily Growth Desk, not old automation
   modules.
3. Run or refresh Account Intelligence from `/accounts/:id`.
4. Save any available private analytics notes manually.
5. Apply the account strategy to Growth Strategy.
6. Open Daily Growth Desk from the strategy handoff.
7. Pick one opportunity that is marked safe enough to handle manually.
8. Generate or write one reply draft.
9. Copy the reply and open the original X post.
10. Mark the opportunity handled.
11. Paste the X reply URL or manually backfill result metrics.
12. Confirm Learning Insights shows the result loop state changing.

## Pass Criteria

- The user can describe the product as a manual, safe X operating workflow.
- The user understands that reply publishing happens manually on X unless X API
  permissions and policy constraints allow a narrower automated action.
- The first useful output is reached without visiting legacy automation pages.
- Result backfill is visible as a learning signal, not a hidden backend detail.
- Any X API limitation is explained as a data boundary instead of a product
  failure.

## Failure Signals

- User asks where to enable auto-commenting or auto-DM as the main value.
- User cannot tell which opportunity to handle first.
- User thinks sample/demo signals are live production data.
- User records a result but cannot see what changed.
- User expects Creator Studio private analytics to be synced automatically.
