# Product Notes Archive

The current product direction is tracked in `docs/product/roadmap.md`,
`docs/product/page-list.md`, and the product-strength plans under
`docs/product/`.

Files in this directory are historical Daily X Queue research and validation
artifacts unless a file explicitly says otherwise. They are useful for context,
but they are not the current first-day product plan.

Current first-day workflow:

- `/start-today` opens the guided Daily Growth Desk path.
- `/exposure-radar` is the primary daily workbench.
- `/content-drafts`, `/content-library`, `/oaf-bots`, and `/handling-list`
  support the manual growth workflow.
- `/daily-x-queue` is downlined compatibility and should not be used as the
  main acceptance or smoke-test route.
