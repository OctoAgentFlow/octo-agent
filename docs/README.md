# Documentation

This public documentation set intentionally keeps only reference material that
is useful to contributors and operators of a self-hosted development instance.

## API

- `api/README.md`: API documentation entry point.
- `api/*.md`: Route-level API notes.

## Database

- `database/tables.md`: Database table overview.
- `database/er-diagram.md`: Entity relationship diagram.

Internal launch notes, deployment runbooks, growth plans, acceptance reports,
and private product planning documents are not part of the public repository.
