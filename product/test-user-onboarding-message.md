## Test Instructions

1. Open https://test.octo-agent.com/daily-x-queue.
2. Log in with your test account.
3. Enter your X handle and product context.
4. Add your target audience, voice preference, and guardrails.
5. Paste one source item, such as a product note, launch note, customer insight, changelog, FAQ, or positioning note.
6. Click Generate 3 drafts.
7. Review all 3 draft cards.
8. Edit one draft if it is close but not right.
9. Reject one draft if it misses the mark and choose a reason.
10. Approve or copy one draft if it is useful.
11. Try again on another day with a new source item.

## What Users Should Try

Try source material that reflects real operating work:

- product update
- launch note
- founder thought
- customer objection
- feature explanation
- positioning note
- short changelog
- FAQ answer
- rough idea you have not turned into a post yet

Try to use the review actions honestly:

- copy drafts you would actually use
- edit drafts that are close
- reject drafts that are not useful
- use reject reasons that explain what went wrong
